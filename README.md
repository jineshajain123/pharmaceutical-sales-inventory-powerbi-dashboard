# 💊 Pharmaceutical Sales & Inventory Analytics Dashboard
📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze pharmaceutical sales and inventory performance using a dataset containing over **150,000 records**.

The dashboard helps business users monitor revenue, profit, inventory health, supplier performance, stock levels, and product expiry through interactive visualizations.

---
🚀 Features

- Executive Sales Dashboard
- Inventory & Supply Chain Dashboard
- Dynamic KPIs
- Interactive Slicers
- Revenue Trend Analysis
- Product Category Analysis
- Regional Sales Performance
- Inventory Monitoring
- Low Stock Detection
- Product Expiry Tracking

---

🛠 Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling
- Excel

---

📊 KPIs

- Total Revenue
- Total Profit
- Profit Margin
- Average Order Value
- Total Orders
- Inventory Value
- Total Stock
- Low Stock Products
- Expiring Products
- Total Suppliers
